Welcome to the Administrator's Guide

Hello, esteemed Student Committee Members! This guide will help you navigate the admin features of our website. Follow the steps below to effectively manage content on the site.

1. How to Add New News

  1. Login to the website using your admin credentials.
  2. Navigate to the Admin Dashboard.
  3. Click on the "Add News Article" button.
  4. Fill out the Title, Content, and upload a Banner Image.
  5. Click to publish the news.

2. How to Edit Existing News

  • From the Admin Dashboard, find the news article you wish to edit.
  • Click on the button next to the article.
  • Make the necessary changes to the Title, Content, or upload a new Banner Image.
  • Press to save changes.

3. How to Change Text Color and Use HTML Tags

You can enhance your news articles using simple HTML tags:

HTML Tag Description Example
<strong> Makes text bold. This is bold text.
<em> Makes text italic. This is italic text.
<u> Underlines text. This text is underlined.
<font color="red"> Changes text color. This text is red.

To change the color of your text, you can use the <span> tag with the style attribute. For example:

<span style="color: blue;">This text is blue.</span>

Which will display as: This text is blue.

4. How to Write News in HTML

When adding or editing a news article, you can use HTML tags to format your content. Here's how:

1. In the Content field, type your news article using regular text.

2. To add formatting, insert HTML tags directly into the text.

3. Use headings, paragraphs, lists, images, and more to enrich your article.

For example:

<h3>Upcoming Events</h3>
<ul>
    <li>Science Fair - June 5th</li>
    <li>Art Exhibition - June 12th</li>
</ul>

Will display as:

Upcoming Events

  • Science Fair - June 5th
  • Art Exhibition - June 12th

5. How to Update the Progress Report

Keep everyone informed by updating the progress report:

  1. Go to the Admin Dashboard.
  2. Select "Manage Progress Reports".
  3. In each section (Plan, Doing, Done), list items on separate lines.
  4. Click to save your changes.

Additional Tips

  • Use clear and concise language in your articles.
  • Include images to make your news more engaging.
  • Always review your content before publishing.

Thank you for your contribution to keeping our community informed!
